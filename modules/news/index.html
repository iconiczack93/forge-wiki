    News Aggregator

    RSS feed aggregation with Discord integration — polls news sources and posts new articles.

    File: modules/news.py | Lines: ~610

    What It Does

    Polls 20+ RSS sources (WaPo, Reuters, AP, BBC, etc.) every 30 minutes. Deduplicates via SQLite (tracks seen articles, 7-day retention). Posts new articles to 📰-news Discord channel as rich embeds with source branding, category emoji, and clickable links. Supports filtering by category (world, tech, national security, etc.).

    Key Functions

    Function Purpose
    init_news_aggregator(bot, channel_id) Start polling loop, create DB table
    poll_sources() Fetch all RSS feeds, parse, dedupe
    post_article(article, source) Format embed and post to Discord
    _clean_seen_cache() Remove articles older than 7 days
    add_rss_source(name, url, category) Register custom RSS feed

    Configuration

    • Poll Interval: 30 minutes
    • Max Posts Per Poll: 5
    • Sources: 20+ default (WaPo, AP, Reuters, CNN, BBC, etc.)
    • Categories: world, national, politics, business, tech, investigative, national security
    • Embed Colors: Per-source branding (WaPo: #2B2B2B, AP: #E3120B, etc.)

    Dependencies

    • feedparser — RSS parsing
    • aiohttp — async HTTP
    • discord.py — embed formatting
    • Database: news_articles table
