    Classifier Prompt

    Intent classification template with 470+ action definitions.

    File: core/classifier_prompt.py | Lines: ~2000+

    What It Does

    Contains the Haiku classifier's intent definition template. Extracted from bot.py to enable independent versioning. Defines all action types FORGE recognizes (browser, email, calendar, settings, etc.) with examples for each action. Routes natural language messages to the correct handler via JSON classification.

    Key Actions

    Action Group Example Actions
    Browser browser_open, browser_screenshot, browser_tabs, browser_close, browser_read, browser_back
    Screen screen_capture, screen_describe
    App app_open, app_close, app_list
    PC pc_lock, pc_stats, pc_volume, pc_sleep, pc_shutdown, pc_restart
    Email email_inbox, email_search, email_send, email_reply, email_delete, email_mass_delete
    Calendar calendar_today, calendar_add, calendar_update, calendar_delete, calendar_reschedule, calendar_freebusy
    Media media_play_pause, media_next, media_prev, media_volume, media_sort
    Settings settings_view, settings_update

    Key Structure

    Field Purpose
    _ACTION_CLASSIFY_PROMPT Main classification template with action descriptions
    HANDLED_ACTIONS Maps action names to handler functions in cogs
