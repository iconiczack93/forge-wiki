    Finance & Utilities

    FORGE's personal finance, media control, smart home, and PC management systems.

    Finance Module: Personal Financial Advisor

    Tracks spending, income, subscriptions, budgets, and savings goals with AI-powered advice.

    Auto-Categorization

    36+ keyword-based categories: - Food: restaurant, chipotle, doordash, groceries, coffee - Entertainment: Netflix, Spotify, Twitch, concert, movie - Subscriptions: Apify, OpenAI, GitHub Copilot, Cursor, Patreon - Transport: Uber, Lyft, gas, parking, insurance - Tech: Amazon, GPU, monitor, laptop, electronics - Bills: rent, electric, internet, water, mortgage - Health: gym, doctor, pharmacy, therapy, supplements - Shopping: Target, Walmart, Costco, furniture

    Matches first keyword found in transaction description.

    Core Functions

    Function Purpose
    log_transaction(amount, category, description) Log expense (negative) or income (positive)
    get_transactions(limit=100) Fetch recent transactions
    delete_transaction(txn_id) Remove transaction
    get_spending_summary(days=30) Category breakdown + total
    get_monthly_comparison() Month-over-month trends
    check_budget_status() Alert if over budget
    set_budget(category, limit) Define spending caps
    add_recurring(item, category, amount, interval) Subscription/bill tracking
    get_subscription_audit() All subscriptions, renewal dates
    add_savings_goal(name, target, deadline) Goal tracking
    get_financial_advice() AI-powered recommendations (Haiku)

    Discord UI: Finance Dashboard

    Modal dialogs: - Log Expense: amount ($), description - Log Income: amount ($), description - Set Budget: category, limit - Add Goal: name, target amount, deadline

    Persistent buttons: - Log Expense, Log Income, Summary, Subscriptions, Goals, Advice

    Spotify Control: Full Playback Integration

    Control Spotify from Discord via Spotify Web API (spotipy).

    Features

    • Playback: play, pause, next, previous, shuffle, repeat
    • Search: artists, albums, tracks, playlists
    • Queue: add to queue, view queue
    • Liked Songs: play your library
    • Playlists: browse, play, search within playlist
    • Now Playing: current track, artist, album

    Configuration

    SPOTIFY_CLIENT_ID=...
SPOTIFY_CLIENT_SECRET=...
SPOTIFY_REDIRECT_URI=http://localhost:8888/callback

    Scopes: - user-read-playback-state - user-modify-playback-state - user-read-currently-playing - playlist-read-private - user-library-read

    Usage

    from modules.spotify_control import play_playlist, get_now_playing

await play_playlist("workout")
now = await get_now_playing()
→ {"track": "...", "artist": "...", "album": "..."}

    Smart Home Bridge

    Control IoT devices from Discord.

    Supported Devices

    Device Auth Features
    Roku Auto-discover (mDNS) Power, volume, input, app control
    Ring Email + password + 2FA Snapshots, motion alerts, status
    Sonos IP address Play, pause, volume, grouping
    Thermostat API key + token (Ecobee/Nest/Honeywell) Temp get/set, mode changes
    Govee API key (from Govee app) Lights on/off, brightness, color, temp

    Setup

    Roku: Zero setup! Auto-discovers on network. Optional: ROKU_IPS=... to skip discovery.

    Others: Add to .env when ready: 

    RING_EMAIL=your@email.com
RING_PASSWORD=...
SONOS_IP=192.168.1.100
THERMOSTAT_TYPE=nest
THERMOSTAT_API_KEY=...
GOVEE_API_KEY=...

    Service Architecture

    Base class SmartHomeService: - is_configured(): bool - get_status(): dict - execute(command, **kwargs): str response

    • Screenshot: take screenshots, OCR
    • Clipboard: read, write

    Package Tracker Module

    Monitor shipments from Amazon, UPS, FedEx, USPS via tracking APIs.

    Tracking

    • Amazon: order number → delivery status, tracking
    • UPS: tracking number → location, ETA
    • FedEx: tracking number → current status
    • USPS: tracking number → delivery date

    Database

    SQLite package_tracker table: - item_name, carrier, tracking_number - status, last_update, estimated_delivery - created_at

    Torrent Sorting Module

    Organize completed torrent files into library structure.

    Features

    • Auto-detect: completed torrent directory
    • Extract metadata: series name, season, episode (regex parsing)
    • Organize: move into TV/Movies/Archive structure
    • Cleanup: remove duplicate files

    Path Structure

    /media/tv/
  ├── Show Name/
  │   ├── Season 1/
  │   └── Season 2/
/media/movies/
  └── Movie Name (Year)/
